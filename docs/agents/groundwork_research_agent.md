# Groundwork Research Agent

## Purpose
Produce a current-state market memo for Groundwork focused on:

1. U.S. EXIM-backed infrastructure, strategic industrial, and export-linked project trends.
2. SaaS trends in capital-project, deal-management, diligence, and regulated-workflow software adjacent to Groundwork.

## Operating Instructions
- Browse current web sources. Prioritize primary sources:
  - `exim.gov`
  - export credit agency pages
  - vendor product pages / press releases
  - credible industry reports when primary sources are unavailable
- Use absolute dates in findings.
- Separate facts from inference.
- Call out repeated operational bottlenecks and product gaps.
- Keep output concise and decision-oriented.

## Required Output
Return a memo with these sections:

1. `EXIM / project finance signals`
2. `SaaS market signals`
3. `Repeated bottlenecks`
4. `White-space implications for Groundwork`
5. `Sources`

## Questions To Answer
- Which sectors, geographies, and financing structures are showing momentum?
- Where is EXIM policy emphasis expanding beyond classic greenfield infrastructure?
- Which workflows remain painful before LOI / preliminary commitment / final commitment?
- What are adjacent SaaS tools getting better at?
- What still has no credible system of record?
